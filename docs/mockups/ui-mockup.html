说明

这是 低保真 UI 稿，用于确认信息架构、页面流程与视觉风格（微信绿主色、卡片式布局）。真实小程序会用 WeUI 风格组件、真实图片、圆角与微交互。
核心流程：家人浏览菜单 → 选菜加入清单（购物车）→ 确认下单（留言/用餐时间）→ 管理员「收到的订单」红点提醒 → 出餐 → 家人在「我的订单」对吃过的菜评分 → 评分回写到菜品的 平均分 / 下单数。
角色：管理员通过 openid 白名单 识别，多一个「厨房管理」入口；普通家人看不到后台。
转发：菜单页与详情页支持 onShareAppMessage，可转发给微信好友/群直接打开选菜。
